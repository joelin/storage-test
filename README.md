# storage-test

## size
* 1 B = 8 bit
* 1 KB = 1,024 Bytes　
* 1 MB = 1,024 KB(kilobyte)　
* 1 GB = 1,024 MB(megabyte)
* 1 TB = 1,024 GB(gigabyte)
* 1 PB = 1,024 TB(terabyte)
* 1 EB = 1,024 PB(pebibyte)
* 1 ZB = 1,024 EB(ExaBytes)
* 1 YB = 1,024 ZB(zebibyte)
